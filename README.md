# Artist App

A React application built with Vite, TypeScript, Tailwind CSS, and React Router.

## Requirements

- **Node.js** v20 or higher
- **npm**

## Installation

After cloning the repo, install all dependencies

```
npm install
```

## Scripts

### Lint the project

```
npm run lint
```

### Run tests

```
npm run test
```

### Run in development

```
npm run dev
```
